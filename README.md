## 📱 نبذة عن التطبيق

**FOR U FATIMA** هو تطبيق موبايل جميل وسهل الاستخدام مصمم خصيصاً لمساعدة الفتيات على اكتشاف أفضل الأنشطة اليومية بناءً على:
- 🎭 حالتهن المزاجية
- ⚡ مستوى الطاقة لديهن
- ⏰ الوقت المتاح لديهن
- 🎯 أهدافهن الشخصية

---

## ✨ الميزات الرئيسية

### 🎯 استبيان ذكي (30 سؤالاً)
- أسئلة مصممة بعناية لفهم احتياجاتك
- واجهة تفاعلية وممتعة
- شريط تقدم لطيف وسلس

### 🎨 تصميم جميل وأنثوي
- ألوان وردية ناعمة وباستيل
- أزرار دائرية وعناصر ودية
- واجهة متجاوبة بالكامل

### 💡 5 توصيات نشاط مخصصة
- مهارات وتطوير ذاتي
- هوايات وأنشطة إبداعية
- استرخاء ورفاهية
- ترفيه وتسلية
- وأكثر!

### 📹 فيديوهات تعليمية مباشرة
- روابط يوتيوب لكل نشاط
- خطوات تنفيذية مفصلة
- تعليمات واضحة وسهلة

## 🚀 البدء السريع

### المتطلبات
- Flutter SDK (الإصدار 3.0 أو أحدث)
- Dart SDK
- جهاز أندرويد أو محاكي

### التثبيت

1. **استنساخ المشروع:**
```bash
cd "c:\Users\hp\Documents\for u"
```

2. **تثبيت المتعلقات:**
```bash
flutter pub get
```

3. **تشغيل التطبيق:**
```bash
flutter run
```

4. **بناء ملف APK (الإصدار النهائي):**
```bash
flutter build apk --release
```

5. **بناء ملف AAB (لمتجر Google Play):**
```bash
flutter build appbundle --release
```

## ⚙️ التخصيص والتعديل

### تغيير الأسئلة:
عدّل ملف `lib/data/questions_data.dart`

### إضافة أنشطة جديدة:
عدّل ملف `lib/data/activities_data.dart`

### تغيير الألوان:
ابحث عن قيم `Color(0xFFXXXXXX)` في الملفات

### تحسين منطق التوصيات:
عدّل ملف `lib/services/survey_processor_service.dart`
