# مشروع FOR U FATIMA

## نقاط البدء السريع

### 1. التثبيت الأولي
```bash
cd "c:\Users\hp\Documents\for u"
flutter pub get
```

### 2. تشغيل التطبيق على محاكي
```bash
flutter run
```

### 3. بناء APK للإصدار
```bash
flutter build apk --release
```

يتم حفظ ملف APK في:
`build/app/outputs/flutter-apk/app-release.apk`

### 4. تنظيف وإعادة بناء
```bash
flutter clean
flutter pub get
flutter run
```
