## 🎨 Customization Examples

### Change App Colors
Edit any screen file and modify hex colors:
```dart
Color(0xFFFFB6C1)  // Change this
Color(0xFFFF69B4)  // Or this
```

### Add New Questions
Edit `lib/data/questions_data.dart`:
```dart
QuestionModel(
  id: 31,
  question: 'Your question here',
  category: 'category_name',
),
```

### Add New Activities
Edit `lib/data/activities_data.dart`:
```dart
ActivityModel(
  id: 'new_activity',
  title: 'Activity Title',
  // ... other properties
),
```

### Change Recommendation Logic
Edit `lib/services/survey_processor_service.dart` to modify algorithm
